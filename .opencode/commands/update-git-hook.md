---
description: Este comando atualiza automaticamente os nomes dos projetos que estão no monorepo para funcionar corretamente no momento do pré-commit. 
---

Leia a pasta **packages** e **apps** para encontrar o nome dos projetos e packages que estão neste monorepo, e atualize o arquivo **commitlint.config** que está na raiz do monorepo para que o pré-commit funcione corretamente.
